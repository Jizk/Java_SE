css(层叠样式表)： 主要负责页面的样式

编写css代码的方式：
	1、 在<style></style>标签中 编写		只能用于本页面中，复用性较差

		<style type="text\css">
			编写css的代码
		</style>

	2、引用外部的css文件				推荐使用
		1.使用link标签				推荐使用
		
			<link href="css文件的路径" rel="stylesheet"/> 
		
		2.使用style标签
			<style type="text\css">
				@import url("css文件的路径");
			</style>
		
	3、直接在html标签使用style属性编写      只能用于本标签中，复用性差，不推荐使用。

选择器：找到对应的数据进行样式化。
	
	标签选择器：找到所有指定的标签进行样式化
	
		格式：
			标签名{
				样式1;样式2;.......
			}
		
	类选择器：首先要给html标签指定对应的class属性值        
		
		格式：
			.class的属性值{
				样式1;样式2;.....
			}
		
		注意：
			1.class属性值一定不能是数字开头
			2.类选择器的样式要优先于标签选择器的样式
			
	ID选择器：首先要给html元素添加一个id的属性值
		
		格式：
			#id的属性值{
				样式1;样式2;.......
			}
		
		注意：
			1.ID选择器的样式优先于类选择器，优先级最高。
			2.ID的属性值不能以数字开头。
			3.ID的属性值在一个html页面中只能出现一次
			
	交集选择器：对选择器1中的选择器2进行样式化
		格式：
			选择器1 选择器2{
				样式1;样式2;......
			}
	
	并集选择器：对指定的选择器进行统一的样式化
		格式：
			选择器1,选择器2{
				样式1,样式2....
			}
		
	通用选择器：
		格式：
			*{
				样式1;样式2.....
			}
	
	伪类选择器：对元素处于某种状态下进行样式化的
